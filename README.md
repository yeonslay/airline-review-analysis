# airline-review-analysis
> 항공사 리뷰 데이터를 기반으로 **텍스트마이닝(NLP)** 을 수행한 프로젝트입니다.  
> 공개 리뷰 데이터를 활용하여 **워드클라우드**, **감성분석**, **LDA 토픽모델링**을 통해  
> 항공사별 서비스 인사이트를 도출하였습니다.  

---

## 🧭 프로젝트 구성

| 단계 | 파일명 / 폴더 | 설명 |
|------|----------------|------|
| 1️⃣ 크롤링 | `review_crawling.ipynb` | 항공사별 리뷰 데이터를 수집하는 스크립트 |
| 2️⃣ 워드클라우드 | `wordcloud.ipynb` | 리뷰 텍스트 전처리 및 시각화 |
| 3️⃣ 감성 분석 | `review_to_sentence_sentiment analysis.ipynb` | 문장 단위 감성 분류 (긍정/중립/부정) |
| 4️⃣ LDA 토픽 모델링 | `topic_modeling(lda).ipynb` | 항공사별 주요 주제(Topic) 추출 |

---

## 1️⃣ 데이터 수집 (Crawling)

> ⚠️ **주의:** 본 프로젝트의 크롤링 코드는 학습 및 비상업적 연구 목적이며,  
> 원문 데이터는 공개하지 않습니다.

**🧩 기술 스택:**  
`Selenium`, `BeautifulSoup`, `pandas`

**📋 기능 요약**
- 항공사별 리뷰 페이지 자동 탐색  
- 작성일, 본문 텍스트 수집  
- 중복 제거 및 CSV 저장 (`Merged_[항공사].csv`)

---

## 2️⃣ 텍스트 전처리 및 워드클라우드 (WordCloud)

**🧠 형태소 분석기:** `kiwipiepy`  

**🚫 불용어 처리:** `Stopwords` (기본 내장)  

**📋 주요 단계**
1. 한글 텍스트만 추출 (`re` 정규식 사용)  
2. 명사 및 형용사 추출  
3. 항공사별 워드클라우드 시각화  
-> 형용사만, noise 제거, 단어 조합(명사 + 형용사) 다양한 방법으로 시도

---

## 3️⃣ 감성 분석 (Sentiment Analysis)

**📘 분석 방식**
- 감성사전(`SentiWord_info.json`) 기반 **문장 단위** 감성 분류  
- 형태소 분석: `Kiwi`  
- 감정 레이블: `긍정`, `중립`, `부정`

**🎯 목적**
- 한 리뷰 내에 다양한 감성이 존재하기에 문장 단위의 감성 분류 수행
- 기본 감성사전에 **도메인에 맞는** 추가적인 **{단어: 감성}**을 추가

---

## 4️⃣ LDA 토픽모델링 (Topic Modeling)

**🧩 사용 라이브러리:** `gensim`, `pyLDAvis`  
**📋 전처리:** 명사/형용사 기반 토큰화  
**🎯 목적:** 항공사별 주요 주제(Topic) 파악 및 시각화
